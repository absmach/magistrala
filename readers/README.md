# Readers

Readers provide implementations of various `message readers`. Message readers are services that consume normalized (in `SenML` format) Magistrala messages from data storage and expose HTTP API for message consumption.

For an in-depth explanation of the usage of `readers` service as well as API details, see the [Readers guide](https://magistrala.absmach.eu/docs/dev-guide/services/readers/) and the [OpenAPI reference](https://docs.api.magistrala.absmach.eu/?urls.primaryName=api%2Freaders.yaml).
